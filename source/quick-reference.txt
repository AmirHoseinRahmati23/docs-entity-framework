===============
Quick Reference
===============

This page shows the provider syntax for several MongoDB commands.

.. list-table::
   :header-rows: 1
   :widths: 25 75

   * - Command
     - Syntax

   * - | **Set up ``DBContext``**

     - .. literalinclude:: /includes/code-examples/quick-reference.cs
          :start-after: // start-db-context
          :end-before: // end-db-context
          :language: csharp
          :dedent:

   * - | **Find a Document**

     - .. io-code-block::
          :copyable: true

          .. input::
             :language: csharp

             var planet = db.Planets.FirstOrDefault(p => p.name == "Mercury");
             
             Console.WriteLine(planet.ToBsonDocument());

          .. output::
             :language: json

             { "_id" : ObjectId("..."), "name" : "Mercury", "orderFromSun" : 1, "hasRings" : false }

   * - | **Find Multiple Documents**

     - .. io-code-block::
          :copyable: true

          .. input::
             :language: csharp

             var planets = db.Planets.Where(p => p.hasRings == true).ToList();
             
             foreach (var p in planets)
             {
                Console.WriteLine(p.ToBsonDocument());
             }

          .. output::
             :language: json

             { "_id" : ObjectId("..."), "name" : "Jupiter", "orderFromSun" : 5, "hasRings" : true }
             { "_id" : ObjectId("..."), "name" : "Uranus", "orderFromSun" : 7, "hasRings" : true }
             { "_id" : ObjectId("..."), "name" : "Saturn", "orderFromSun" : 6, "hasRings" : true }
             { "_id" : ObjectId("..."), "name" : "Neptune", "orderFromSun" : 8, "hasRings" : true }

   * - | **Insert a Document**

     - .. code-block:: csharp
          :copyable: true

          db.Planets.Add(new Planet() { name = "Pluto", hasRings = false, orderFromSun = 9 });
          db.SaveChanges();
